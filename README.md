# Smart Doc Tracker

Gestore di documenti e scadenze personali con OCR e reminder automatici.  
Progetto personale di studio realizzato in **React + TypeScript + Next.js + Supabase**.

## Tech Stack

- **Next.js 15** – framework React moderno
- **TypeScript** – tipizzazione sicura
- **Tailwind CSS** – styling rapido e modulare
- **shadcn/ui** – libreria di componenti UI
- **Supabase** – backend-as-a-service (auth + DB + storage)
- **Vercel** – hosting e deploy

## Funzionalità previste (MVP)

- Upload manuale dei documenti
- Salvataggio dei metadati e scadenze
- Reminder automatici
- Dashboard con ricerca e filtri
- OCR base (Tesseract.js o API esterna)

## Setup locale

Clona la repo e installa le dipendenze:

```bash
git clone https://github.com/lauradionisi/smart-doc-tracker.git
cd smart-doc-tracker
npm install
npm run dev
```
